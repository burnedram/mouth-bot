# MouthBot

Rythm bot is dead, long live rythm bot!

## Commands

### `/play`

Plays a YouTube video, playlist or search query.

### `/join`

Moves the bot into your channel.

### `/stop`

Disconnects the bot.

### `/queue`

Show the currently playing song and queue.

### `/skip`

Skips songs.

## Running

`npm run watch`
